# Summary

This is an early version of the release of Dexter HD. 
Some 3D printed parts may be missing.
We will be updating this over the next few weeks to complete it.
We are also working on a video manual for instructions on how to build it.

The filenames contain the number of copies and the version number.
For example, 'FlexSplineAttach_x3_v1', is version 1 and needs 3 copies printed.
